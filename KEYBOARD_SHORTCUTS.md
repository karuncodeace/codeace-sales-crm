# Keyboard Shortcuts Guide

This document lists all available keyboard shortcuts in the Sales CRM application.

## Navigation Shortcuts (G + Letter)

Press `G` followed by another key to navigate to different pages:
`
| Shortcut | Action | Description |
|----------|--------|-------------|
| `G` + `D` | Go to Dashboard | Navigate to the main dashboard |
| `G` + `L` | Go to Leads | Navigate to the Leads management page |
| `G` + `P` | Go to Prospects | Navigate to the Prospects page |
| `G` + `T` | Go to Tasks | Navigate to the Tasks page |
| `G` + `A` | Go to Appointments | Navigate to the Appointments/Bookings page |
| `G` + `R` | Go to Revenue | Navigate to the Revenue page (Admin only) |
| `G` + `O` | Go to Proposals | Navigate to the Proposals page |
| `G` + `U` | Go to Profile | Navigate to the User Profile page |

## Search & Focus Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `K` | Focus Search (Dashboard) | Focus the search bar on Dashboard page |
| `Ctrl/Cmd` + `K` | Focus Search (Global) | Focus the search bar from anywhere |
| `F` | Focus Filter | Focus the filter/search input on list pages (Leads, Tasks, Prospects) |
| `ESC` | Unfocus/Close | Close modals, unfocus inputs, or close dropdowns |

## Quick Actions

| Shortcut | Action | Description | Context |
|----------|--------|-------------|---------|
| `N` | New/Create | Open create modal | Context-aware: Opens "Add Lead" on Leads page, "Add Task" on Tasks page |
| `B` | Book Meeting | Open book meeting modal | Only on Lead detail pages |
| `S` | Save | Save form data | When in a form context |
| `E` | Edit | Edit current item | Context-aware (when available) |
| `D` | Delete | Delete current item | Only on Lead detail pages (when available) |

## UI Controls

| Shortcut | Action | Description |
|----------|--------|-------------|
| `H` | Toggle Sidebar | Collapse or expand the sidebar |
| `?` | Show Help | Display this keyboard shortcuts help dialog |

## Notes

- **Shortcuts are disabled** when typing in input fields, textareas, or contenteditable elements
- **Navigation shortcuts** show a toast notification when activated
- **Context-aware shortcuts** (like `N` for New) adapt based on the current page
- **Modifier keys** (Ctrl/Cmd/Alt) are generally ignored except for specific shortcuts like `Ctrl/Cmd + K`
- All shortcuts work globally except `K` which only works on the Dashboard page

## Usage Tips

1. **Quick Navigation**: Use `G` + letter combinations for fast page navigation
2. **Search Quickly**: Press `K` on Dashboard or `Ctrl/Cmd + K` anywhere to search
3. **Create Items**: Press `N` on any list page to quickly create new items
4. **Get Help**: Press `?` anytime to see all available shortcuts
5. **Close Things**: Press `ESC` to close modals, dropdowns, or unfocus inputs
